# HR_analytics_project
Project Description:-
This is a HR Analytics project where I analyzed employee data to understand attrition and other important factors.
What I Did in This Project?
Imported HR dataset (CSV file)
Cleaned the data (removed null values and duplicates)
Organized and prepared data for analysis
Analyzed employee attrition
Compared salary, department, and job role
Created interactive dashboard in Power BI.
Used charts like bar chart, donut chart,column chart, and KPIs etc.
Tools Used:-
Power BI
Excel
Key Insights:
Some departments have higher attrition
Employees with low salary leave more
Experience affects employee retention
This project helped me understand how to analyze HR data and create dashboards for better decision making.
